DD-DIT Channel Report

Google Ads API Integration — Design Document (Internal Tool)
Project: works-site / dddit  |  Date: July 2026  |  Contact: ddditchannel@gmail.com

1. Purpose

디디딧(DD-DIT) YouTube 채널 팀이 사용하는 비공개 내부 워크스페이스입니다. Google Ads API를 통해 본인 Google Ads 계정의 프로모션 캠페인 비용·노출·조회·클릭 데이터만 읽어와 채널 보고 대시보드에 표시합니다. CPV·CPM 등 효율 지표를 YouTube Analytics 데이터와 함께 검토합니다.

  • 대상 URL: https://works.mansejin.com/dddit/report/ (noindex, team-only)
  • API access: read-only — no campaign create/edit/delete
  • Audience: internal team members only (not public SaaS)

2. Architecture

[ Google Ads Account 144-794-4603 ] | | Google Ads API (search / metrics read) v [ NAS Docker: works-api ] ← OAuth refresh token, developer token (server-side .env) | | HTTPS REST (private) v [ works.mansejin.com/dddit/report/ ] ← team browser (no API keys in frontend)

3. Data Flow

  1. Team opens channel report page or clicks Refresh.
  2. Backend calls POST /api/dddit/youtube/report/ads/sync (on-demand).
  3. works-api queries Google Ads API for campaign metrics (last 30 days).
  4. Results cached in NAS data/youtube/ads-sync.json and merged with manual notes.
  5. Report UI shows promotion table: cost, impressions, views, CPV, efficiency text.
