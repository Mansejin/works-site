# works-site

개인·업무용 페이지. `mansejin.com` 메인 도구함과 분리된 서브도메인.

- 라이브: https://works.mansejin.com
- GitHub Pages + Custom domain `works.mansejin.com`

## DNS (가비아)

| 타입 | 호스트 | 값 |
|------|--------|-----|
| CNAME | `works` | `Mansejin.github.io` |

저장 후 GitHub **Settings → Pages → Custom domain**에 `works.mansejin.com` 입력.

## 배포

`main` 브랜치 push 시 GitHub Pages 자동 배포.
