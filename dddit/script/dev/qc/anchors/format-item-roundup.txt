# N개 아이템 라운드업 포맷 앵커 (구조·속도 — 톤 앵커와 별도)

소스·콘셉트는 **카테고리**로 구분: 다이소, 쿠팡, 테무, 데스크테리어, 무인양품 등.
아이템 블록 구조·호흡은 소스와 무관하게 동일.

## 오프닝 패턴 (6행 전후)
- 안녕하세요, 디디딧입니다
- 시즌·상황 공감 한 줄
- "그래서 준비했습니다" 전환
- 콘셉트 한 줄 (소스·테마·N가지)
- 가격·효과 한 줄
- "바로 살펴볼까요" 진입

## 아이템 블록 패턴
[제품명]          ← 대괄호 단독 행 (필수)
한 줄 정의 (…입니다)
특징·사용감 2~4호흡
구매 팁·주의·가격 (선택)

## 클로징 패턴 (N개 아이템)
- 오늘 소개해드린 [콘셉트·제품군명] 어떠셨나요?
- … / 댓글 / 구독과 좋아요
- 그럼 오늘 소개해드린 제품의 **정보**를 정리해드리며… (단일 리뷰의 장단점·평점과 다름)
- 감사합니다!

## 톤 차이 (협찬 심층 대비)
- 더 빠른 호흡, 제품당 5~8행
- "품절", "가성비", 가벼운 감탄 OK
- 심층 비교·장문 논증 금지
