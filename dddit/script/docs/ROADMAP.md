# 디디딧 시나리오 머신 — 로드맵

## 최종 목표

**제품 제원 + 리뷰 방향 + 엑셀(구글 시트) 콘티**를 한 흐름으로 완성하는 도구.

| 산출물 | 형식 | 설명 |
|--------|------|------|
| 제품 브리프 | 구조화 입력 | 제원·방향·비교·주의점 |
| 파트 구성 | 라인업 | 프롤로그 → 본문 → 총평 |
| 콘티 대본 | TSV 5열 | 대본 · 장면 · 사이즈 · 자막 · 코멘트 |

---

## Phase 1 — 시나리오·콘티

- [x] **공기청정기 서치** (Google Search 그라운딩, 2026 여름·가격대 필터)
- [x] 서치 결과 → 브리프 자동 적용
- [x] 파트별 대본 생성 (라이트·프로 모델 분리)
- [x] 호흡·행 길이·바이트 검증
- [x] TSV보내기 (구글 시트 콘티)
- [x] 제품 브리프 UI (제원 템플릿 + 리뷰 방향)
- [ ] 브리프 → 프롬프트 품질 검증 (공기청정기 등 카테고리별 튜닝)
- [ ] 파트별 «콘티 체크리스트» (장면·사이즈 누락 경고)

### 작업 흐름 (현재)

```
1. 서치     2026 여름 공기청정기 (Google Search) → 후보 5개
2. 브리프   «브리프에 적용» → 제원·방향 자동 채움
3. 파트     AI 초안 → 확정
4. 콘티     파트별 대본 → 표
5. TSV      구글 시트
```

## Phase 2 — 국내 IT·가전 리서치 (예정)

아이템 발굴·제원 수집 자동화. **API 키 + 백엔드 또는 MCP** 필요.

| 기능 | 설명 | 입력 예 |
|------|------|---------|
| 상세페이지 분석 | 제조사·쇼핑몰 URL → 스펙·특징 추출 | 삼성·LG·쿠팡 상세 |
| 제원 DB화 | 카테고리별 필드 자동 매핑 | CADR, 필터, 소음… |
| 유튜브 리뷰 분석 | 영상 URL/검색 → 요약·팩트·누락 포인트 | 국내 IT 리뷰 채널 |
| 아이템 탐색 | 키워드·카테고리로 후보 제품 리스트 | «1인 가구 공기청정기» |

### 예시 제품: 공기청정기

- 카테고리: `공기·환기`
- 핵심 제원: CADR, 권장 평수, 필터 비용, 소음, 1인 거실 체감
- 앱 내 **«공기청정기 예시 불러오기»** 로 브리프 구조 체험 가능

## Phase 3 — 통합 워크스페이스 (장기)

- 리서치 결과 → 브리프 자동 채우기
- 유튜브·상세페이지 출처 링크 보존
- 팀 공유용 프로젝트 파일 (JSON export/import)
- 버전 관리된 프롬프트 + 브리프 스냅샷

## 기술 메모

- **현재**: 정적 HTML + Gemini API (브라우저 직접 호출)
- **Phase 2 제약**: 상세페이지·유튜브 크롤링은 CORS 때문에 로컬 서버 또는 별도 API 프록시 필요
- **권장**: Phase 1 콘티 품질 확정 후 `start.bat` 로컬 서버 + 리서치 모듈 추가
